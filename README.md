# Shopping Application for Trainee
Мобильное приложение для определения нывыков разработки мобильных приложений.

## Технологии
 * Kotlin
 * Kotlin Coroutines (Flow)
 * Retrofit2
 * Kotlinx Serialization
 * Glide
 * Hilt
 * Paging3
 * MVVM
 * Architecture components
 * View binding
 * Material Design
